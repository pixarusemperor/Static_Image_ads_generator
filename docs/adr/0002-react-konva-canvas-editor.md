# 0002. React-Konva Canvas Editor

We evaluated various open-source image editors, whiteboard canvases, and canvas engines. We decided to use React-Konva (Konva.js) as the foundation for the Canvas Editor. It provides clean React-based state integration and seamless JSON serialization of visual layers (shapes, images, text blocks), making it easy to synchronize edits between the user, the backend Programmatic Assembler, and the AI agent.
